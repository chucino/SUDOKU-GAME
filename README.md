# Sudoku Game

A Sudoku game built with Vue 3, TypeScript and Vite.

This project implements a classic 9×9 Sudoku game where players fill numbers from 1 to 9 without repeating in each row, column, and 3×3 box. The game supports multiple difficulty levels (Easy, Medium, Hard), generates valid Sudoku boards using backtracking and constraint propagation, and ensures a unique solution for each puzzle.
